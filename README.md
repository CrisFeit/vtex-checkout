# Vtex Checkout

##  Features
* Webpack
* Babel
* ESLint
* Sass
* PostCss
* ITCSS Architecture

## Requirements
* Node.js >=12.0.0
## Install
```bash
  npm install
```
## Guide
* Files path for development mode
```
http://localhost:8080/files/checkout6-custom.js
http://localhost:8080/files/checkout6-custom.css
```
## Comand
- Development
```bash
npm run dev
```
- Production
```bash
npm run build
```
