# Vtex Checkout

##  Features
* Webpack
* Babel
* Sass
* PostCss

## Requirements
* Node.js >=12.0.0
## Install
```bash
  npm install
```
## Guide
* Files path for development mode
```
http://localhost:8080/
│
└───│   checkout6-custom.js
    │   checkout6-custom.css
```
## Comand
- Development
```bash
npm run dev
```
- Production
```bash
npm run build
```
